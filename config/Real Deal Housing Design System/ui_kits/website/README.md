# Website UI kit

Recreation of the Gallery White marketing site from `Real Deal Housing OS/web/src/app/(site)/`.

- `index.html` — interactive click-through (bottom-right switcher, or real CTAs: "View listings →", "New launch · DLF Westpark")
- `WebsiteHome.jsx` — home: hero, launch banner, projects, properties, pillars, testimonial
- `WebsiteListings.jsx` — buy/rent grid with type toggle
- `WebsiteWestpark.jsx` — the 10-section launch landing (eyebrows 01–10, multi-state map card, verified-facts ledger, FAQ disclosures)
- `WebsiteMobile.jsx` — 390px phone view with the two-segment sticky CTA (hides when the enquiry section scrolls into view)

Real Westpark photography from the codebase is used where the emails used it; everything else stays an honest dashed placeholder, per brand rules.
